## 🚀 Proyecto destacado — uti-bunna

Sistema de **carpooling inteligente** para la comunidad Riwi. Arquitectura híbrida con doble base de datos:

- **PostgreSQL + PostGIS** → usuarios, autenticación, rutas geográficas y matching espacial
- **MongoDB Atlas** → registro de matches aceptados entre drivers y pasajeros
- **OSRM API** → cálculo automático de rutas desde la ubicación del driver hasta la sede
- **JWT + bcrypt** → autenticación segura con roles `driver` / `passenger`
- Algoritmo de matching en 4 filtros secuenciales: rol → turno → bounding box → distancia < 1km

```
Stack: Node.js · Express · PostgreSQL · MongoDB · PostGIS · JWT · Docker
```
